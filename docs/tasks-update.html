Tasks: update

Requires authorization

Update the duration of a task lease. Try it now.

Request

HTTP request

POST https://www.googleapis.com/taskqueue/v1beta2/projects/project/taskqueues/taskqueue/tasks/task

Parameters

Parameter name

Value

Description

Path parameters

project

string

The project under which the queue lies.

task

string

taskqueue

string

Required query parameters

newLeaseSeconds

integer

The new lease in seconds.

Authorization

This request requires authorization with at least one of the following scopes (read more about authentication and authorization).

Scope

https://www.googleapis.com/auth/taskqueue

https://www.googleapis.com/auth/taskqueue.consumer

Request body

In the request body, supply a Tasks resource.

Response

If successful, this method returns a Tasks resource in the response body.
